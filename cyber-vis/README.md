# 🚀 CyberVis - Визуализация кибератак в реальном времени

![CyberVis Dashboard](docs/screenshot-dashboard.png)

**Дипломный проект** - система мониторинга и визуализации кибератак на основе анализа журналов событий.

## 📋 Основные возможности

### 🔍 Детектирование атак
- **Сканирование портов** - обнаружение массового сканирования
- **Brute-force атаки** - множественные попытки входа
- **Географические аномалии** - входы из разных стран за короткое время

### 🗺️ Визуализация в реальном времени
- **Интерактивная карта мира** с маркерами атак
- **Динамические графики** статистики
- **Лента событий** с WebSocket обновлениями

### 📊 Анализ и отчетность
- **История всех атак** с фильтрацией
- **Экспорт данных** в CSV/JSON форматах
- **Статистика** по типам атак и странам

## 🚀 Быстрый старт

### Вариант 1: Запуск с Docker (рекомендуется)


# Клонируйте репозиторий
git clone https://github.com/ваш-аккаунт/cyber-vis.git
cd cyber-vis

# Запустите с Docker Compose
docker-compose up -d

# Откройте в браузере
open http://localhost:8000

### Вариант 2: Локальная установка
# Установите зависимости
poetry install

# Запустите сервер
poetry run uvicorn src.cyber_vis.app.main:app --reload

# Откройте в браузере
open http://localhost:8000
